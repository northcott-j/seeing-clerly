Overview

Due to the Clery Act of 1990, all universities must publish a minimum set of crime statistics each year - and this web tool allows you to see and interact with this data. The graph is initially set to display the reported cases of rape from five of the major universities in Boston. To interact with the application, you may select which crime you want to view, then choose your university. The state, city, and student population fields will help filter the results available in the school input. If the graph gets stuck, simply hit "Refresh" and it will check to make sure it drew all included data.

The data is sourced from the US Dept. of Education, and was cleaned and uploaded into a database hosted on mLab; the plotting uses ChartJS's javascript library. See the repository's readme for a full list of credits. *This is still in beta, so the data may not be 100% accurate, nor will the site be perfect - but we're working on it!*
